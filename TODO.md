# 🗺️ 功能开发计划 (Roadmap)

## 🚀 高优先级 (High Priority)
- [ ] **自定义域名绑定 (Custom Domains)**  
  为 Pro 用户提供 CNAME 接入 (如 `resume.zhangsan.com`)，极大提升专业度。
- [ ] **高级数据分析 (Advanced Analytics)**  
  不仅仅是浏览次数。增加“访问来源” (LinkedIn/GitHub) 和“地理位置”分布，帮助用户了解谁在看简历。

## ✨ 体验优化 (User Experience)
- [ ] **PDF 版本历史与回滚 (Version History)**  
  保留最近 10 次生成的 PDF。如果新版本编译错误，允许用户一键回滚到上一个可用版本。
- [ ] **同步状态通知 (Notifications)**  
  当 Overleaf 编译失败或同步中断时，通过邮件或 Server酱/微信 通知用户。

## 🛠️ 进阶功能 (Advanced Features)
- [ ] **简历多版本管理 (A/B Testing)**  
  支持同一个项目输出多个 PDF (如 `resume-en.pdf`, `resume-cn.pdf`)，方便针对不同岗位投递。
- [ ] **社交预览图自动生成 (OG Image)**  
  当用户把链接分享到 Twitter/微信 时，自动截取 PDF 第一页作为预览图。

## 💰 商业化 (Monetization)
- [ ] **Stripe 支付集成**  
  接入真实的支付流程，实现自动订阅和续费。
