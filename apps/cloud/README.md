# OverLink Platform (SaaS)

This directory contains the source code for the centralized SaaS version of OverLink.

## Structure

-   **`web/`**: The Next.js Application (Vercel).
    -   Frontend Dashboard
    -   API Endpoints
-   **`worker/`**: The Python Worker (GitHub Actions).
    -   Runs Playwright
    -   Uploads to Cloudflare R2

## Getting Started

See individual `README.md` files in each subdirectory for setup instructions.
